# pattern-maker
A website that creates a custom conical corset pattern

Inspired by a youtube video showing how to make one physically.

## Requirements

### Primary
* Learn how to do web apps/sites
* Use git and github
* Transfer Pattern maker from Processing Java to javascript
* Display pattern on screen
* Basic UI
* PNG pattern export

### Secondary
* Nice UI design
* Export PDF/SVS

### Third
* Reactive UI
* Host site

## Technology (not final)
* React.js for code
* Bootstrap for UI
* Canvas for drawing the pattern
